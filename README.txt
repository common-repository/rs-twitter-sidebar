== Description ==

Another twitter plugin :). Difference is the hover effect. It can be in the left or right side. Twitter plugin will be shown on hovering over the icon.

After getting huge success in [Opencart](http://www.opencart.com/index.php?route=extension/extension/info&extension_id=19473&filter_username=rosinghal), I have made this free plugin for Wordpress.

== Installation ==

1. Upload `rs-twitter-sidebar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Settings > RS Twitter Sidebar for settings.

== Frequently Asked Questions ==

= What should I enter in Twitter widget ID? =

You have to enter your Twitter widget ID here. To get your widget ID goto https://twitter.com/settings/widgets. You will get a button to create new widget.
